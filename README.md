# FSL-Manager

## Usage
- Select the downloaded WINMM.dll
- Select the GTA 5 Enhanced folder
- Click Install/Uninstall FSL to copy/delete to the game folder

## Settings
- You can edit FSL's settings.ini directly through the manager

## Extras
**Clear Local Save**: You can delete the local FSL saves
**Delete Local Profile**: You can delete local gta 5 profiles that sometimes cause errors

**Additional notes**: FSL Manager saves the last used paths in %AppData%/FSL Manager
